# Inference Prompt Resources

This directory contains prompt templates used for metadata inference and derived annotation generation within the GEOMeta pipeline.

## Included Tasks

- derive_agegroup_from_age.md
  Infer standardized age-group labels from raw age annotations.

- infer_perturbation_type.md
  Infer perturbation categories from perturbation descriptions.

- infer_sex_from_tissue.md
  Infer sex specificity from anatomical or tissue annotations.

## Notes

These prompts support rule-guided metadata augmentation and standardization during GEOMeta processing workflows.
